# Actuarial Python Learning

My journey learning actuarial mathematics with Python.

## About

This repository contains my work learning actuarial concepts through programming, including:
- Interest theory and time value of money
- Annuity calculations and present value
- Survival models and life tables
- SOA exam practice problems

## Files

- `actuarial_ch2.py` - Chapter 2: Interest Theory examples
  - Interest rate conversions (i, d, Î´, v)
  - Annuity calculations (immediate, due, continuous)
  - SOA practice problems
- `test_actuarial.py` - Basic actuarial tests and examples

## Setup

Install the actuarialmath package:
```bash
pip install actuarialmath
```

## Technologies

- Python 3.10+
- actuarialmath package
- NumPy, SciPy, Matplotlib
